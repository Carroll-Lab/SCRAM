SCRAM

SCRAM is lightweight Python package for aligning small RNA reads to one or more reference sequences and producing publiciation-quality images.

Installation

Unpack to a location of you choice. If your Python 2.7 path differs from /usr/bin/python, modify line 1 of scram.py in a text editor.
