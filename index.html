SCRAM

SCRAM is lightweight Python package for aligning small RNA reads to one or more reference sequences and producing publiciation-quality images.

Developed by Stephen Fletcher @ the laboratory of Prof. Bernie Carroll, University of Queensland

Installation

Unpack to a location of your choice. Use directly :

python scram.py req1 req 2 [opt]

Or install in your path :

  1. If your Python 2.7 path differs from /usr/bin/python - modify line 1 of scram.py to reflect your python path.
  2. Make scram.py executable : chmod +x /path/to/scram.py
scram.py req1 req 2 [opt]

Required Python packages:

matplotlib
numpy

File Format

Reference File - DNA sequences only (AGCT) - FASTA format
Sequence File - Collapsed reads - DNA sequences only (AGCT) - FASTA format
Post-processing of FASTQ reads to collapsed FASTA foramt can be carried out using the FASTX-Toolkit from the Hannon Lab.
An example of the required format: head seq1.fa
>1-607041 TCGGACCAGGCATCATTCCCC >2-202886 TCGGACCAGGCTTCATACCCC >3-71446 TCCCAAATATAGACAAAGCA

Usage

scram.py analysis_type reference_file [-h] [-s1 SEQ_FILE_1] [-s2 SEQ_FILE_2] [-s3 SEQ_FILE_3] [-s4 SEQ_FILE_4] [-nt SRNA_LEN] [-f FILE_NAME] [-seq_list SEQ_LIST] [-min_read MIN_READ_SIZE] [-max_read MAX_READ_SIZE] [-min_count MIN_READ_COUNT] [-win SMOOTH_WIN_SIZE] [-ylim YLIM] [-no_display] [-split] [-pub] [-V]
